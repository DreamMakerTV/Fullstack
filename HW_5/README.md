# 🎸 Rock Portal

## 📖 Про проєкт

**Rock Portal** — це багатосторінковий веб-сайт, створений для справжніх поціновувачів рок-музики, важких рифів та глибоких текстів. Проєкт об'єднує однодумців та слугує зручним майданчиком для пошуку свіжих музичних новин, історії легендарних гуртів та спілкування.

## ✨ Функціональні можливості та структура

Проєкт має продуману навігацію та складається з наступних сторінок:

- **Головна (`index.html`):** Привітання користувачів та вступна інформація про світ рок-культури.
- **Про нас (`about.html`):** Інформація про команду розробників-фанатів з України та контактні координати.
- **Статті (`articles.html`):** Стрічка музичних новин, історії створення інструментів та добірки найкращих рок-фестивалів.
- **Галерея (`gallery.html`):** Інтерактивна секція «Легенди Року» з картками культових гуртів (Metallica, Iron Maiden, Kiss, Led Zeppelin та ін.) і посиланнями на Вікіпедію.
- **Контакти (`contacts.html`):** Контактні дані та зручна форма зворотного зв'язку.
- **Система автентифікації:** Окремі сторінки для входу (`login.html`), реєстрації (`register.html`) та відновлення пароля (`forgot-password.html`), включаючи UI для авторизації через Google та Facebook.

## 🛠 Технології

- HTML5 (семантична розмітка)
- CSS3 (стилізація, Flexbox/Grid)
